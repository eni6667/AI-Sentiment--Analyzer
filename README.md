# AI Sentiment Analyzer

A simple Artificial Intelligence project that uses Natural Language Processing (NLP) to classify text sentiment.

## Features

- Detects positive and negative emotions
- Uses a pretrained AI model
- Simple command line interface

## Technologies

- Python
- Hugging Face Transformers
- PyTorch

## How to Run

Install dependencies:

pip install transformers torch

Run:

python sentiment_analyzer.py
